## Getting Started

### System Requirements
- Docker Engine
- [TARS-Outpost](https://github.com/quadseed/TARS-Outpost) (for Auto Recording Hook)

### Set environment variable
Rename `.env.sample` to `.env` and fill in the required information

### Run Server
```shell
$ docker-compose up
```
